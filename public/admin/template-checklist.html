üîß Veel Voorkomende Problemen

Sliders werken niet:

  1. Check browser console voor errors
  2. Verify: typeof jQuery !== 'undefined'
  3. Verify: typeof $.fn.slick !== 'undefined'
  4. Check selector: $('.hero-slider').length > 0

Menu niet dynamisch:

  1. Check console: "‚úÖ Dynamic menu injected"?
  2. Test selector in console
  3. Check database: show_in_menu = true?

Pagina's niet vindbaar:

  1. Check slug in database (geen leading slash!)
  2. Test API direct: /api/page/[slug]
  3. Check status = 'published'
