# Tripex Template Preview Images

## Required Images

### Category Preview (800x1200 portrait)
- `tripex-preview.png` - Overview of entire Tripex template

### Page Previews (1200x800 landscape)  
All 25 pages need preview images with naming: `{page-slug}-preview.png`

## How to Generate

Use a screenshot tool or headless browser to capture:
1. Full page screenshots at 1200x800
2. Crop/resize to exact dimensions
3. Compress to max 300KB per image
