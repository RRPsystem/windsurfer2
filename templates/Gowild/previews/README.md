## How to Generate

Use a screenshot tool or headless browser to capture:
1. Full page screenshots at 1200x800
2. Crop/resize to exact dimensions
3. Compress to max 300KB per image

Tools:
- Browser DevTools (F12) → Device toolbar → Screenshot
- Playwright/Puppeteer for automated screenshots
- Online tools like screenshotmachine.com
